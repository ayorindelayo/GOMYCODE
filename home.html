HTML

HTML: Hypertext markup language.
It is a programming language that enables the creation of the framework that defines websites, it provides a link between the developer and the browser.

CSS

CSS: Cascading style sheet.
It serves as the palette of web development as it enables design of web pages, it generally coordinates size, positioning, color, font amongst others. It gives the HTML code displayed on a web page “life” with color and effects therefore creating a more user-friendly web page.

JavaScript

JavaScript is a programming language that is used to manipulate and control the behavior of web pages, in association with multiple libraries. JavaScript makes a web page interactive, providing a user-friendly experience.
